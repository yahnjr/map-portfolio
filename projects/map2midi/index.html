    • Turning Geospatial Data into (Questionably) Beautiful Sounds

    Skills used: Python, Javascript, web application building, libraries, front and back end (full-stack) development, self-led projects, UI/UX design

    A personal project exploring the relationship between the fundamental Tobler's First Law of geospatial science and music. In this project, I wanted to see if turning the coordinates geographers are familar with and mapping them to midi files would produce beautiful sounds. The results are interesting, sometimes melodic, and often dischordant and horrifying, but it was a fun tool to create. I first created a script to utilize geopandas to turn the x-coordinate of a spatial point into the timing of a note, and the y-coordinate into the note being played. I added some visualisation tools, includng a world map that shows the note being played in real time, and some various instruments, or soundfonts, to listen to them with. I learned a lot about both spatial data and musical files, as well as many obscure libraries of Python and Javascript (including how hard it is to integrate them into a Conda based environment).

    The full sized web app can be found here.
