Columbia Slough Watershed Nature Access Application

Skills used:

Created in partnership with the Columbia Slough Watershed Council and a supported by a federal grant, this application is intended to accompany print maps to give residents information about areas where they can enjoy the natural areas of the Columbia Slough. Accessibility and inclusivity are crucial to the functionality of this application. The application supports four language modes (English, Spanish, Ukranian and Russian) and allows for the filtering of locations based on accessibility features.

The full sized web app can be found at their website (full page version here).
