    • Campview: A New Way To View Campsites Before Booking

    Skills used:

    Born from a perhaps overly ambitious project to create a 3D model of all the Northwest Area campsites, this application allows users to view a photosphere style (think Google Street View) image of their campsite before deciding where to pitch a tent. I learned a lot about application development, data storage, and the time consuming act of data collection while working on this project, and I also got to spend some time in the great outdoors.
