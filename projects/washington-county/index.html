Custom Tools for Washington County Public Survey Application

Skills used:

While working with 3J Consulting and Washington County, OR, assisted in supporting the GIS division of the county Survey Department. A part of this support was modernizing and upgrading the "Survey Explorer" tool, which is used by county staff and the public to access county data. For this purpose, two widgets were created and added to the Experience Builder application hosted on the public facing website.

Incremental Measurements: The first widget created was a custom measure widget, to extend the functionality of the "out of the box" measure widget. To activate it, click the ruler icon below the search. It can be used to measure distances and areas. For distances, this creates labels and a table of each segment of measurement, as well as a bearing. It also includes several units of measurement used by surveyors, such as rods (66 feet).

Document Viewer The second widget was designed to create a snapshot view of documents accessible via the application in a static, shareable format. It can be opened with the download icon below the search bar. The widget watches the number of features for activated layers visible in the current view, and when that crosses below a defined threshold, prompts the user to create a document view. This will automatically create a shareable HTML document that includes a map screenshot as well as embedded versions of all relevant PDFs related to features.

The full sized web app can be found at the Washington County website.
